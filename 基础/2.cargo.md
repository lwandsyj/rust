1. Cargo是Rust的构建系统和包管理器

> 创建 new

        cargo new learn1  // 创建learn1 项目

> 安装 install 软件包

        cargo install 

> 运行 run 

        cargo run 

> 文档 doc

        cargo doc 

> 构建 build

        cargo build

        cargo build --release
